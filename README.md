## 📌 Descrição

Dashboard interativo de acordo com o primeiro trimestre do ano de 2019 de um supermercado. Feito totalmente com Python, desde o tratamento dos dados até plotagem dos gráficos.


## Tecnologias Usadas

- Python 3.12
- Straemlit 1.42
- Pandas 2.2.3
